Прогнозирование цен на акции

О программе

Это веб-приложение разработано для прогнозирования будущих цен на акции с использованием передовых методов машинного обучения. Оно использует исторические данные о ценах на акции для прогнозирования цены закрытия следующего дня.

Технологии и методология

Основой нашей модели прогнозирования является сеть долгой краткосрочной памяти (LSTM), тип рекуррентной нейронной сети, подходящей для анализа последовательных данных. Сети LSTM особенно эффективны в решении проблемы долгосрочных зависимостей, что делает их идеальными для прогнозирования временных рядов, таких как цены на акции.

Приложение построено на Python с использованием веб-фреймворка Flask для бэкенда и использует Keras с бэкендом TensorFlow для построения и запуска нейронной сети. Данные поступают в реальном времени из Yahoo Finance через библиотеку yfinance, что обеспечивает актуальность предсказаний.

Как это работает:

  • Сбор данных: Исторические данные о закрытии цен выбранной акции загружаются за последний год.
  • Предобработка данных: Данные нормализуются с использованием MinMaxScaler, чтобы обеспечить подачу данных в масштабе, подходящем для операций нейронной сети.
  • Обучение модели: Модель LSTM обучается на этих исторических данных, изучая закономерности и аномалии в движении цен на акции.
  • Прогнозирование: Модель делает прогноз на основе данных о последних 60 днях закрытия цен.

Такой подход позволяет нашему приложению предоставлять прогнозы с разумной степенью точности, делая его ценным инструментом для инвесторов и трейдеров, стремящихся к осознанным решениям.
